Best of 3 Challenge
===================
Create a script that flips a coin three times and reports the winner to the console.

* [Demo](http://jsbin.com/jiraqi/1/edit?console)
* [Start](http://jsbin.com/jiraqi/edit?js,console)

### Sample Output

```
    Heads Wins
      Heads: 2
      Tails: 1
        JSON Results {\"heads\":2,\"tails\":1}
            
```
